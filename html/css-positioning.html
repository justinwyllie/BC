CSS Positioning

As well as styles (such as colours, text effects and so on) CSS can be used to control positioning on a page.

There are 4 main values for the position property:

  • absolute
  • fixed
  • relative
  • static

The 4 main types are explained below. However; you should also be aware that there is a propery 'float' which makes elements 'float' outside of the normal flow. There are also systems for organising layout such as Flexbox and Grid. These will be covered separately.

Static

This is the default setting. In this mode elements on a page just flow, basically in the order they appear in the html. You should understand that some elements are block elements and some are inline. A block element has a line-break underneath it. <div> is a block element. <span>s are inline.
